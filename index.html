Window 和 Screen 对象详细对比

核心区别说明

Window 对象：代表浏览器窗口，包含窗口大小、位置、滚动等信息

Screen 对象：代表用户整个屏幕，包含屏幕分辨率、颜色深度等硬件信息

关键差异：Window是软件层面（浏览器），Screen是硬件层面（显示器）

// Window 对象 - 用于窗口操作 window.innerWidth // 浏览器窗口内容区域宽度 window.outerWidth // 浏览器窗口总宽度 window.scrollX // 水平滚动距离 window.open() // 打开新窗口 // Screen 对象 - 用于屏幕检测 screen.width // 屏幕总宽度 screen.height // 屏幕总高度 screen.availWidth // 可用屏幕宽度（去除任务栏） screen.colorDepth // 屏幕颜色深度
响应式设计中的实际应用
// 检测屏幕类型 function detectScreenType() { const screenWidth = screen.width; const windowWidth = window.innerWidth; console.log('屏幕实际宽度：', screenWidth); console.log('浏览器窗口宽度：', windowWidth); if (screenWidth <= 768) { console.log('移动设备屏幕'); } else if (screenWidth <= 1024) { console.log('平板设备屏幕'); } else { console.log('桌面设备屏幕'); } } // 居中显示弹窗 function centerPopup() { const popupWidth = 400; const popupHeight = 300; // 使用 screen 计算屏幕中心位置 const left = (screen.width - popupWidth) / 2; const top = (screen.height - popupHeight) / 2; window.open('popup.html', 'popup', `width=${popupWidth},height=${popupHeight},left=${left},top=${top}`); }
